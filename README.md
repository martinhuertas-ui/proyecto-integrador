# 📚 Proyecto Integrador - Sistema de Gestión Académica

Sistema desarrollado en Java que permite la gestión de estudiantes, cursos y calificaciones, aplicando principios de programación orientada a objetos (POO).

---

## Tecnologías utilizadas

* Java
* IntelliJ IDEA

---

## Funcionalidades

* Registro de estudiantes
* Creación y gestión de cursos
* Asignación de notas
* Uso de interfaces para definir comportamientos
* Aplicación de programación orientada a objetos

---

## Conceptos aplicados

* Clases y objetos
* Encapsulamiento
* Herencia
* Polimorfismo
* Interfaces

---

## Estructura del proyecto

* `src/Model` → Clases principales del sistema (Estudiante, Curso, Nota, etc.)
* `src/Interfaces` → Interfaces como Mostrable, Reportable, Calificable
* `src/Main` → Clase principal para ejecutar el programa

---

## Cómo ejecutar el proyecto

1. Clonar el repositorio:

```
git clone https://github.com/martinhuertas-ui/proyecto-integrador.git
```

2. Abrir el proyecto en IntelliJ IDEA

3. Ejecutar la clase `Main`
